# Ai-Based-Chatbox

FINTECH POLICY DOCUMENTS - For RAG Hands-On

This folder contains 3 policy documents for the Fintech Knowledge Copilot notebook:

1. kyc_policy.txt       - KYC requirements for Tier 1, 2, 3 wallets
2. dispute_resolution.txt - Dispute and chargeback procedures
3. product_faqs.txt     - Wallet and card product information

HOW TO USE:
1. Upload this entire folder (fintech_docs) to your Google Drive
2. Place it in "My Drive" (e.g. My Drive/fintech_docs)
3. Open the Fintech_Knowledge_Copilot notebook in Google Colab
4. Run Step 2 to mount Drive - the notebook will look for this folder
5. If your folder is in a different path, update DOCS_DRIVE_PATH in the notebook

Required files: kyc_policy.txt, dispute_resolution.txt, product_faqs.txt
